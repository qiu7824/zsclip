# ZSClip 0.7.4

本次版本重点修复了分组菜单稳定性和高 DPI/缩放下的主列表布局问题，并继续整理本地发布内容。

## 主要更新

### 1. 修复分组菜单异常退出
- 修复“新建分组后，右键常用短语程序会退出”的问题
- Tab 分组菜单不再在打开前提前切换当前 Tab
- 只有真正选中分组筛选项后，才会应用目标 Tab 的过滤状态
- 去掉分组菜单关闭后强行恢复搜索焦点的副作用

### 2. 修复缩放后列表图标和位置错位
- 主列表的左侧图标、置顶图标、快捷按钮、文本留白进一步统一到主布局计算
- 去掉一批未按 DPI 正确放大的固定上限
- 改善不同缩放比例下图标大小、位置和文本区域不一致的问题

### 3. 本地发布物整理
- 版本号更新到 `0.7.4`
- 中英文 README 的发布页链接同步更新到 `0.7.4`
- 本地 `release` 目录按 `0.7.4` 重新整理
