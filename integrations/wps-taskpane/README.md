# 剪贴板 WPS 任务窗格插件

这是剪贴板的 WPS 任务窗格插件。

## 功能

- 在 WPS 功能区显示“剪贴板 / 打开剪贴板”入口。
- 使用 WPS `CreateTaskPane` 打开 `http://127.0.0.1:38473/office/wps/taskpane`。
- 显示复制记录和常用短语，并自动刷新。
- 支持把文本或图片插入到当前 WPS 文档光标位置。

## 使用

- 启动剪贴板。
- 在设置页插件中启用 `WPS 任务窗格`。
- 在 WPS 中打开插件，点击“打开剪贴板”。
- 如果窗格无法加载，确认本机地址 `127.0.0.1:38473` 可访问；WPS 接口只允许本机请求。
